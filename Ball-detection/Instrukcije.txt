Instrukcije za instalaciju OpenCV preko Anaconde..

1. Skini miniconda - netreba ti anaconda (manja je ali dovoljno)

2. Kada ti se instalila pokrenes apk i pišeš slijedece komanda

pip install opencv-python matplotlib filterpy 
**Ako nemas tu komandu skini python 3.12

conda create -n name 
name - stavi koji god hoces dosiv npr.

conda activate name

i gg nakon toga samo se cd u folder i klasik

python ball-detect.py
